Sintaxis-CSS

Selector: El elemento al que vamos a aplicar estilos.

Propiedad: Lo que vamos a cambiar

Valor: El nuevo valor que va a tener la propiedad

Propiedad:Valor -> Se le denomina declaración Al conjunto de selector+declaracion(es) se le denomina regla

Ejemplo

                
                    
                    selector{
                        
                        propiedad1:valor;
                        propiedad2:valor;
                        propiedadN:valor;
                        
                    }
                
                En la última propiedad no es necesario poner punto y coma ';' ya que solo se usa como separador de propiedades. Aunque es una buena práctica hacerlo con todas.
