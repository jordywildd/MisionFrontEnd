Selectores elementales

                No suelen ser muy utilizados, solo recomendables para casos puntuales.
                Existen 2 tipos, que a su vez se dividen en subtipos.
                Selectores Simples
                Selectores elementales
                Selector Universal -> *
                
                    *{
                        background-color: lightcoral;
                    }
                
                Selector de tipo o etiqueta -> Nombre de la etiqueta.
                
                    h1{
                        background-color: aliceblue;
                    }

Selectores de atributo

            Existen 3 tipos:
            id -> id del elemento
            clase -> clase del elemento
            otros atributos.
            
            
Selectores de atributo - Selector por id.
            

            No es recomendable usarlo ya que uno de los principios de CSS es que los estilos sean reutilizables.
            
                #ID{
                    propiedad:valor;
                }
                     
                     
Selectores de atributo - Selector por clase.

                     Es el más recomendable de utilizar para aplicar estilos con css. Una de sus ventajas es que es facilmente reutilizable, solo basta con poner la misma clase a todos los elementos deseados para que se aplique el estilo.
                     
                         .clase{
                             propiedad:valor;
                            }
                        
                        
                            Los 2 tipos de selectores, id y clase son case sensitive, por lo que hay que escribir ambas iguales. Por ejemplo title y Title no serían lo mismo.

Selectores de atributo - Otros atibutos

                        Usado para otro tipo de atributos que no son ni id ni clases. Para esto se usan []
                        
                            [href]{
                                
                            }
                            [atributo]{
                                propiedad: valor;
                            }
                            [href="https://www.google.com/"]{
                                propiedad: valor;
                            }
                        
                        Otra manera sería:
                        
                            
                            [atributo=valor]{
                                
                            }
                        
                        otra manera con la que se puede comprobar si el atributo empieza por ese valor sería:
                        
                            [atributo^= valor]{
                                
                }
                [href^="color"]{
                    
                }
            
            la manera en la que se aplica un estlo a un atributo que contenga en cualquier parte de su valor lo que se pone como valor en el selector es:
            
                [atributo*=valor]
                [href*=roj]
            
            la manera en la que se le aplica el style al atributo en el que el valor termina con esa palabra es:
            
                [href$=GB]
                [atributo$=ultima  del valor]
            
            y por último, una manera que permite asignar el estilo a la palabra tenga o no guion
            
                [atributo|=valor]

Subtitulo

Para aplicar estilo podrían ponerse 1 por uno parrafo por parrafo, podría crear la clase text1, para el siguiente text2, text3 y text4

                Y de esa manera aplicar el código de la siguiente manera:
                
                    .text1{
                        background-color: darkkhaki;
                    }
                    .text2{
                        background-color: darkkhaki;
                    }
                    .text3{
                        background-color: darkkhaki;
                    }
                    .text4{
                        background-color: darkkhaki;
                    }
                
                Pero la mejor manera, para cumplir el principio de la programación don't    repeat yourself, es la siguiente:
                
                    .text1, .text2, .text3, .text4{
                        background-color: darkslateblue;
                    }
